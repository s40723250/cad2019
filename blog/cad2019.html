week2

創建github倉儲並且練習github指令 1. 創立並且加入課程倉儲名稱為Cad2019 , 然後勾選lnitialize this repository with a README

1.先將Y:\portablegit\bin\sh.exe 改名為 sh_rename_for_solvespace.exe

4.進入資料夾cd solvespace>cd extlib>cd libpng>mkdir build>cd build

5.執行cmake .. -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release mingw32-make

6.找到檔案 libpng.dll.a 更名, 更名名為 libpng_static.a

7.回到 solvespace 目錄,建立 build 目錄後進入 build 資料夾

8.在執行指令cmake .. -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release mingw32-make
